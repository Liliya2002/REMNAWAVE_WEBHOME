# Style Guide — VPN Webhome

Цель: футуристичный, строгий стиль с акцентом на IT-безопасность — тёмный фон, неоновые акценты, чёткая типографика.

1) Цветовая палитра
- Background: `#0B1020` (глубокий тёмный)
- Surface: `#0F1724` (панели/карточки)
- Brand (primary): `#6366F1` (индиго / неоновый)
- Accent (cyan): `#06B6D4` (техно-неон)
- Muted text: `#94A3B8` (светло-серый)
- Success: `#10B981`, Danger: `#EF4444`

2) Типографика
- Основной шрифт: `Inter`, fallback: system UI
- Заголовки: жирность 700, крупные размеры (h1 ~48-56px)

3) Токены (CSS переменные)
- Использовать `src/design-tokens.css` для runtime-переменных.

4) Компоненты
- Кнопка: `primary` (фон `brand`, подсветка `neon`), `ghost` (прозрачная с обводкой)
- Hero: крупный заголовок + CTA
- Карточки: поверхность с лёгкой границей `surface` и небольшим размытием фона

5) Пример использования в Tailwind
- `bg-background text-slate-200` для основной страницы
- `bg-surface border border-slate-800 rounded-lg p-6` для карточек

6) Иконки и графика
- Использовать простые геометрические фигуры и тонкие неоновые акценты. SVG с минимальной палитрой.

Дальше: я добавил CSS-токены и базовые компоненты `Button` и `Hero` в `frontend/src/components`.
